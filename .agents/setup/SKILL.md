# Setup Workflow

Verify tool access before starting real work.

## Commands

| Command | Skill | Description |
|---------|-------|-------------|
| `/setup-preflight` | `skills/preflight.md` | Run interactive preflight checklist |

## When to Use

- First time using the agent workflows
- After adding new capabilities (e.g., cluster access, GitLab CLI)
- When a workflow fails with a tool access error
